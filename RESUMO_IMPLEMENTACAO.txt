PROBLEMA ORIGINAL:
-----------------
Toda vez que o sistema era fechado e reaberto, voltavam todos os dados de 
exemplo que estavam hardcoded no código. Mesmo após excluir OS, clientes ou
técnicos, ao reiniciar o sistema tudo voltava ao estado inicial.

SOLUÇÃO IMPLEMENTADA:
---------------------
✓ Banco de dados SQLite para persistência total de dados
✓ Todos os dados são salvos automaticamente em tempo real
✓ Dados são carregados do banco ao iniciar o sistema
✓ Exclusões são permanentes (não voltam mais)
✓ Sistema inicia vazio (sem dados de exemplo fictícios)

ARQUIVOS CRIADOS/MODIFICADOS:
------------------------------
1. DatabaseManager.java (NOVO)
   - Classe completa para gerenciamento do banco de dados SQLite
   - Métodos CRUD para OS, Clientes, Técnicos e Logs
   - Conexão e inicialização automática do banco

2. HackPanel.java (MODIFICADO)
   - Removido loadSampleData() com dados fictícios
   - Adicionado loadDataFromDatabase() para carregar dados reais
   - Todos os métodos CRUD agora salvam no banco de dados:
     * openOSDialog() → DatabaseManager.saveOS()
     * editSelectedOS() → DatabaseManager.saveOS()
     * finalizeSelectedOS() → DatabaseManager.saveOS()
     * deleteSelectedOS() → DatabaseManager.deleteOS()
     * addClient() → DatabaseManager.saveCliente()
     * deleteClient() → DatabaseManager.deleteCliente()
     * addTech() → DatabaseManager.saveTecnico()
     * deleteTech() → DatabaseManager.deleteTecnico()

3. sqlite-jdbc-3.44.1.0.jar (NOVO)
   - Driver JDBC para banco de dados SQLite

4. compilar-executar.bat (NOVO)
   - Script para compilar e executar o sistema
   - Baixa driver SQLite automaticamente se necessário

5. criar-pacote.bat (NOVO)
   - Cria JAR único com tudo incluído para distribuição

6. testar-banco.bat (NOVO)
   - Verifica se tudo está configurado corretamente

7. README.md (NOVO)
   - Documentação completa do projeto

8. GUIA_RAPIDO.md (NOVO)
   - Guia rápido de uso

LOCALIZAÇÃO DO BANCO DE DADOS:
-------------------------------
Caminho: %USERPROFILE%\.techsuite\techsuite.db
Exemplo: C:\Users\WINDOWS\.techsuite\techsuite.db

ESTRUTURA DO BANCO:
-------------------
Tabelas criadas automaticamente:
  - ordens (ordens de serviço completas)
  - clientes (cadastro de clientes)
  - tecnicos (equipe técnica)
  - logs (registro de atividades)

COMO USAR:
----------
Para desenvolvimento:
  1. Execute: compilar-executar.bat
  2. Ou: java -cp ".;sqlite-jdbc-3.44.1.0.jar" HackPanel

Para distribuição:
  1. Execute: criar-pacote.bat
  2. Distribua: TechSuitePro.jar
  3. Execute: java -jar TechSuitePro.jar

-------
Para fazer backup:
  1. Feche o sistema
  2. Copie o arquivo: %USERPROFILE%\.techsuite\techsuite.db
  3. Guarde em local seguro

PARA RESTAURAR:
---------------

  2. Substitua o arquivo techsuite.db pelo backup
  3. Reabra o sistema
