✅ O Que Foi Corrigido

🎉 PROBLEMA RESOLVIDO!

Antes: Toda vez que você fechava o sistema, voltavam todos os dados de exemplo. Mesmo após excluir, ao reabrir tudo voltava.

Agora: Banco de dados real! Tudo é salvo permanentemente. Dados excluídos não voltam mais!

🎯 Como Usar

1️⃣ Compilar e Executar

Duplo clique no arquivo:

compilar-executar.bat

2️⃣ Fazer Login

Use as credenciais:

Usuário: admin
Senha: admin123

3️⃣ Trabalhar

Crie OS, Clientes, Técnicos. Tudo é salvo automaticamente!

✨ Funcionalidades

  • Criar OS → Salva no banco automaticamente
  • Editar OS → Atualiza no banco em tempo real
  • Finalizar OS → Salva status e data de conclusão
  • Excluir OS → Remove permanentemente do banco
  • Adicionar Cliente → Salva no banco
  • Excluir Cliente → Remove permanentemente
  • Adicionar Técnico → Salva no banco
  • Fechar Sistema → Dados persistem
  • Reabrir Sistema → Carrega dados reais

💾 Banco de Dados

📍 Localização:

C:\Users\WINDOWS\.techsuite\techsuite.db

Este arquivo contém TODOS os seus dados. Faça backup regularmente!

📦 Backup

⚠️ IMPORTANTE!

Para fazer backup:

  1. Feche o sistema
  2. Copie o arquivo: techsuite.db
  3. Guarde em local seguro (pendrive, nuvem, etc)
